# Dataset

## Source
**Flowers Recognition** dataset from Kaggle:  
https://www.kaggle.com/datasets/alxmamaev/flowers-recognition

## How to Download
1. Visit the [Kaggle dataset page](https://www.kaggle.com/datasets/alxmamaev/flowers-recognition)
2. Download `flowers.zip`
3. For Google Colab: Upload to your Google Drive root folder
4. For local setup: Extract into this `data/` directory

## Structure After Extraction
```
data/
└── flowers/
    ├── daisy/       (~764 images)
    ├── dandelion/   (~1052 images)
    ├── rose/        (~784 images)
    ├── sunflower/   (~733 images)
    └── tulip/       (~984 images)
```

## Specifications
- **Total Images**: 4,317
- **Classes**: 5 (Daisy, Dandelion, Rose, Sunflower, Tulip)
- **Format**: JPEG
- **Resolution**: Variable (~320x240 average)
